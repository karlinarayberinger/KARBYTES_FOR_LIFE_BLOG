MICRO_DIRECTORY_DEFINITION

This web page defines the term “micro directory” (within the context of the body of intellectual property named karbytes).

A “micro directory” is a partition of the website named Karbytes For Life Blog dot WordPress dot Com which has the following characteristics.

CHARACTERISTIC_0: Each “micro directory” is referenced via exactly one hyperlink on the (one and only) “macro directory” web page of the aforementioned website.

(A list of hyperlinks which refers to every “micro directory” in karbytes is displayed on the aforementioned “macro directory” web page at the following Uniform Resource Locator: https://karbytesforlifeblog.wordpress.com/raw_github_files_macro_directory/ ).

CHARACTERISTIC_1: Each “micro directory” has exactly one uniquely-corresponding public GitHub repository on the GitHub account whose username is @karlinarayberinger.

(The Uniform Resource Locator to the aforementioned GitHub profile is as follows: https://github.com/karlinarayberinger/ )

CHARACTERISTIC_2: Each “micro directory” can be entirely reproduced out of the source code files and media files in its corresponding GitHub repository.

(Each “micro directory” top-level directory web page displays exactly one hyperlink pointing to that micro directory’s respective GitHub repository. Click on the code button at the top right corner of the GitHub repository web page to download a zip file containing all of the files in that GitHub repository).

CHARACTERISTIC_3: All of the “micro directories” in the aforementioned “macro directory” contain all of the public source code files and all of the public media files which constitute the following two (public) websites (which means that the entirety of karbytes can be reproduced by downloading each of the zip files whose hyperlinks are listed on the aforementioned “macro directory” web page (and each of those zip files contains all of the files of a particular corresponding GitHub repository uniquely associated with exactly one of the aforementioned “micro directories” (and each “micro directory” displays a hyperlink pointing to its uniquely-corresponding GitHub repository)):

primary_website: Karlina Object dot WordPress dot Com

(primary_website_url: https://karlinaobject.wordpress.com/ )

secondary_website: Karbytes For Life Blog dot WordPress dot Com
