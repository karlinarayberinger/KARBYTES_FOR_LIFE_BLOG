RAW_GITHUB_FILES_MACRO_DIRECTORY

The following directory web pages each feature hyperlinks which point to all of the raw files inside of that directory web page’s associated public karbytes-created GitHub repository. Each of those GitHub repositories has a corresponding web page which is part of a website which karbytes created (and the Uniform Resource Locator (URL) for that corresponding web page is highlighted cyan).

Each directory web page which is referred to on this web page is the top-level directory of exactly one “micro directory”. A formal definition of the term “micro directory” is provided at the following Uniform Resource Locator: https://karbytesforlifeblog.wordpress.com/micro_directory_definition/

(Note that micro-directories whose index numbers are highlighed "mint green" (#22bf71) are assumed to be in their finalized states. Meanwhile, micro-directories whose index numbers are highlighted "lavender" (#b36bff) are assumed to be updated in the future).
